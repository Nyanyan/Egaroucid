## ブレンド式

合法手上で正規化したPolicy Networkの方策とEgaroucid for Consoleの方策を、次の式でブレンドします。

```text
ブレンド方策 = (Policy Networkの方策)^α × (Egaroucid for Consoleの方策)^(1 - α)
```

α=0.0はEgaroucid for Consoleの方策のみ、α=1.0はPolicy Networkの方策のみを使います。Egaroucid側の方策生成レベルは21です。

## 対戦の定義

1回の対戦では、1つのXOT局面から色を入れ替えた2局を行います。2局における同一プレイヤー視点の石差を平均し、その符号で対戦の勝敗を決めます。平均石差が正なら勝ち、0なら引き分け、負なら負けです。

勝率は次の式で計算します。

```text
勝率 = (勝ち数 + 0.5 × 引き分け数) / 対戦数
```

したがって、表中の「20対戦」は40実対局に相当します。

## 実行時間と資源使用量

対戦は再開機能を使って5区間に分けて実行しました。各区間の実行時間の合計は約2時間15分です。

最終区間で2秒ごとに測定した値は次の通りです。メモリは対戦プログラム単独ではなく、PC全体の使用量です。

| 項目 | 平均 | 最大 |
| --- | ---: | ---: |
| CPU使用率 | 81.5% | 100.0% |
| GPU使用率 | 7.5% | 23.0% |
| PC全体の使用メモリ | 約103.0 GiB | 約103.6 GiB |
| GPU使用メモリ | - | 約1.6 GiB |
| 子プロセス数 | - | 111 |

共有GPU推論サーバーは最終区間で5,250局面を4,392回のバッチにまとめました。1バッチ当たりの平均局面数は1.20、最大局面数は4で、推論時間の合計は16.43秒でした。

- 対戦結果は900件で、対戦IDの重複と欠落はありません。
