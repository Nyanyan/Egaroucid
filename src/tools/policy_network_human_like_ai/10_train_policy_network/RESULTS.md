# WTHORポリシーネットワーク学習結果

関連issue: #613

実験日: 2026-07-17

## 要点

以前の実験には、検証データで最も良かった途中エポックのモデルを採用した結果が混ざっていました。
その比較は条件が揃っていないため、今回の採用判断には使いません。

今回の実験では、すべての条件を50エポックまで学習し、50エポック終了後のモデルだけを評価しました。
変えた条件はモデルの幅と層数だけです。

## データセット

入力データは、WTHORから展開した次のboard dataです。

```text
$EGAROUCID_DATA/train_data/board_data/records1
```

seed `613` でWTHOR全局面サンプルをシャッフルし、局面サンプル単位で学習データ、検証データ、テストデータに分割しました。

| 用途 | 局面サンプル数 |
| --- | ---: |
| 学習 | 6,428,225 |
| 検証 | 803,528 |
| テスト | 803,529 |
| 合計 | 8,035,282 |

## 評価方法

人間の着手との一致率は、合法手だけを残したポリシーネットワーク出力で計算しました。
1位一致率は、WTHORで実際に打たれた手が合法手内で最も確率の高い手だった割合です。
上位n手一致率は、実際に打たれた手が合法手内の確率上位n手に入った割合です。

## 採用モデル

採用モデルは `幅512、4層` です。

採用理由は、固定条件で50エポックまで学習したモデル同士を比べたとき、検証データ1位一致率が最も高かったためです。
検証データ1位一致率は57.398%、テストデータ1位一致率は57.471%でした。

今回試した範囲では、幅を768以上に増やす、または6層に増やす構成は、固定条件下では性能が上がりませんでした。
